# 📘 JavaScript Problems for Practice

This repository contains **2500 JavaScript problems** scraped from [Edabit](https://edabit.com).
It is designed to help you practice and improve your JavaScript skills without requiring a paid subscription.

---

## 📖 About

* Problems are organized into folders based on ranges:

  * `001-500`
  * `501-1000`
  * `1001-1500`
  * `1501-2000`
  * `2001-2500`
* Each problem is a **standalone JavaScript file**.
* File names are descriptive and hint at the task.
* You can practice at your own pace.

---

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/rizwanbhuttoo/javascript-problems-for-practice.git
```

### 2. Open in Your Editor

Open the cloned repository in your favorite code editor (e.g., VS Code).

### 3. Start Solving

* Navigate to the folder containing the problems you want to solve.
* Open the corresponding JavaScript file.
* Read the problem description carefully.
* Try to solve it on your own.

---

## ▶️ Running Your Code

Make sure you have Node.js installed.

Run a JavaScript file using:

```bash
node filename.js
```

## 💡 Tips for Effective Practice

* **Learn by Doing**
  Focus on solving problems yourself before looking at solutions.

* **Use Resources**
  If you're stuck, search online or use AI tools for hints and explanations.

* **Stay Consistent**
  Practice regularly to build and strengthen your skills over time.

* **Collaborate**
  Share your solutions and discuss them with others to learn different approaches.
